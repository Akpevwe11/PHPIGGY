## SIMPLE MVC FRAMEWORK IN PHP

This is a simple MVC framework in PHP. It is a simple implementation of the MVC pattern in PHP. It is not a full-fledged framework like Laravel or Symfony. It is just a simple implementation of the MVC pattern in PHP.

## Features

- Routing
- Controllers
- Models
- Views
- Database
- Error Handling
- CSRF Protection
- Flash Messages
- Form Validation
- Pagination
- File Upload
- Authentication
- Authorization
